# Grace Cottage

Vite, React, TypeScript, Tailwind CSS landing page for Grace Cottage sober living in Decaturville, Tennessee.

## Development

Install dependencies and run locally:

```bash
npm install
npm run dev
```

Build static files for hosting:

```bash
npm run build
```

Upload the contents of `dist/` to Hostinger.

## Application Form

The resident application currently runs as a static front-end form. It validates required fields and guides visitors to call or email Grace Cottage directly. It does not send or store application answers.
